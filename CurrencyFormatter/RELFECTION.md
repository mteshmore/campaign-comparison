### Which GenAI tools did you use and for which parts of this assignment?
I used ChatGPT for all of my queries. I relied on it to solidify my understanding of Redux, speed up UI code for the frontend, and generate fake data for testing.

### Give one specific example where AI saved you significant time.
AI was especially helpful for repetitive tasks, like generating fake data. I also used it frequently to find the right component names for Material UI assets. Additionally, I needed to deepen my understanding of Redux, so I leveraged AI to walk me through it step by step.

### Give one specific example where AI output was wrong or suboptimal and you had to correct it.
When working with Material UI libraries, AI often suggested very complicated solutions. To address this, I narrowed my searches with specific keywords to get simpler, more optimal designs, which saved time and allowed me to focus on functionality.

### How do you decide what to delegate to AI vs. write yourself?
I usually delegate error-catching, bug fixing, or research for the most suitable library components to AI. I also use it to generate backend code based on the logic I’ve already set in my head. Everything else, especially the core logic and problem-solving, is my own work.

### How do you verify AI-generated code is production-ready?
I verify AI-generated code by writing and running test cases, reviewing the code myself, and having other users test it in real scenarios.
